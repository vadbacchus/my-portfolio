разное

StartingHTML-Template Bootstrap 4

Особенности шаблона:

  • оставлен и немного переписан таск "img", который был назначен для сжатия картинок;
  • в комплекте с шаблоном идет scss-библиотека Font Awesome, которую можно кастомизировать и подключить к проекту;
  • также к проекту можно подключить собранную мной с помощью сервиса Fontello и переписанную под sass легкую бибиотеку наиболее используемых в интернете иконок - CIP (Custom Icon Pack)
  • индексный файл содержит дополнительные полезные метаданные, которые также отвечают за оформление панели браузера на маленьких устройствах.

StartingHTML-Template (Bootstrap 4) - это стартовый шаблон для разработки сайтов, который в значительной степени базируется на стартовом шаблоне OptimizedHTML-4 известного фронт-энд разработчика АлексеяКлиманова. Целью создания этого шаблона было добавления необходимого функционала, которого не было в OptimizedHTML-4, который позволил бы быстро и удобно разворачивать проекты, в которых используется Bootstrap 4.

StartingHTML-Template SMART GRID

Мой подход к разработке на этом шаблоне состоит в том, что в проекте должен быть только код, который непосредственно используется в проекте. Например случаи, когда подключена jQuery, с которой используется только процентов 10, являются неприемлемыми. Поэтому я рекомендую использовать только те сss и js-библиотеки, которые являются легкими и которые будут максимально задействованы в проекте, а также не забывать о написании популярных фишек на чистом js. Я ни в коем случае не навязываю свой подход к разработке, пользователь шаблона может делать с ним, что угодно. В исходном коде шаблона также лежит scss библиотека Bootstrap 4, scss библиотека иконок Font Awesome, которые можно в любой момент подключить.

StartingHTML-Template (Smart Grid) - это стартовый шаблон для разработки сайтов, отличающийся тем, что включает в себя Smart Grid - одну из самых удобных технологий для быстрого создания адаптивных лэйаутов, которую создал программист ДмитрийЛаврик. Верстка с помощью этой технологии позволяет создать чистый и понятный код, который неперегруженный тонной классов в html и который очень легко поддерживать в дальнейшем. В шаблоне Smart Grid уже настроен в gulpfile.js и сгенерирована sass библиотека, которая подключена к проекту. Другие мелкие особенности шаблона перекочевали напрямую с StartingHTML-Template (Bootstrap 4).
